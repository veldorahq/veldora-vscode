CHANGES IN THIS VERSION
-----------------------
  [+] 10 new snippets for Phase 3 auth/security directives
      v-csrf    →  @csrf
      v-auth    →  @auth ... @endauth
      v-guest   →  @guest ... @endguest
      v-admin   →  @admin ... @endadmin
      v-unless  →  @unless ... @endunless
      v-forelse →  @forelse ... @empty ... @endforelse
      v-for     →  @for ... @endfor
      v-while   →  @while ... @endwhile
      v-php     →  @php ... @endphp
      v-dump    →  @dump($var)

  [+] Updated syntax grammar
      - Highlights @csrf, @auth, @endauth, @guest, @endguest as keywords
      - Highlights @admin, @endadmin for admin-only blocks
      - Highlights @php, @endphp, @dump as special PHP blocks
      - Highlights @unless, @endunless, @forelse, @endforelse, @empty

  [+] Production-ready README.md with badges and snippet reference table
  [+] MIT LICENSE file added
  [+] repository field added to package.json (Marketplace ready)
